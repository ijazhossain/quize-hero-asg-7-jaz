Q.1: Write the differences between forEach, map, find and filter methods?

forEach() method takes every element of an array and executes an operation on each array element which is being provided as a function. This method does not return a new array but it changes the existing array.
map() method iterates every element of the given array to perform a task and after that this method returns a new array.
find() method is used to find any element form an array which fulfils a given condition, instead of returning an array this method only returns the first element which satisfies the condition or returns undefined if so such element is not found in the array.
filter() method is used to create an array with those elements which fulfil a given condition provided to that method by a function. This method returns an array of element or elements which satisfy the provided condition.If so such element is not found then it returns an empty array.
